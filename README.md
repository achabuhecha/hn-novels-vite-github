# hn-novels client-small (Alternative Lightweight App) / 备用轻量版

[English](#english) | [中文](#chinese)

<a name="english"></a>
## English

### Introduction
The `client-small` project is a secondary lightweight frontend instance, structurally identical to `client-lite`. It typically serves as a backup deployment or a testbed for different configurations/modifications without affecting the main lite version.

### Tech Stack
- **Framework**: Vue 3 + Vite
- **Styling**: Tailwind CSS

### Usage
Usage is identical to `client-lite`.

```bash
npm install
npm run dev
```
